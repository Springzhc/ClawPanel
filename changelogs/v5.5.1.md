# ClawPanel v5.5.1

发布时间：2026-04-30

## OpenClaw 配置兼容修复

- 不再把 `agents.defaults.model` 兼容反写为根级 `model`，避免新版 OpenClaw 报 `Unrecognized key: "model"`
- 写入 `openclaw.json` 前统一清理根级 `model` / `sessionDir`，覆盖面板配置和 panel-chat runtime 场景
- 补充回归测试，确保模型配置保存后仍保留 `agents.defaults.model`，但不会写出新版 schema 不接受的 legacy 字段

## 梦境模式保护

- 面板启动前的 OpenClaw 自动修复会保留 `plugins.entries.memory-core.config.dreaming`
- 增加测试覆盖 `dreaming.enabled`、`timezone`、`frequency`，避免梦境模式配置被误清理

## 发行链路收口

- Lite 发行线停止维护后，安装、更新和发布元数据统一指向 GitHub Releases
- 更新文档、安装脚本默认版本和 release metadata 到 `5.5.1`
